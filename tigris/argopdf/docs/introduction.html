    Introduction
 

ArgoPDF is a module for ArgoUML, which is an open source UML modelling tool. The aim of this module, is to provide a possibility of generating pdf reports of data contained in ArgoUML. This module is based on iText library.

  The development of ArgoPDF

ArgoPDF is developed by Dzmitry Churbanau during the Google Summer of Code 2007 under the steerage of his mentor Ion Saving.
