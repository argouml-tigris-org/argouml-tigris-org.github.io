This part describes the user interface of ArgoPDF.


  Overview of the Window

Figure "Overview of the ArgoPDF window" shows the ArgoPDF dialog window, which is displayed when user goes to Tools -> ArgoPDF.
The window is divided into three parts:

  • Top of the window, where user can select the path where report will be saved or enter it manually.
  • Options part which is divided into tabs ("General" and "Title page" tabs), where user can select the report options and the report contents.
  • Bottom of the window, where "Generate" and "Cancel" buttons are situated.

Figure "Overview of the ArgoPDF window"


At the top of the window there is an input field with ". . ." (browse) button. By using this button user can select the path where report will be saved and enter the name of the file. See figure "Choose the path".

Figure "Choose the path"


The path and the name of the report can also be entered manually.
The path where report will be saved and the name of the report the user can see in the appropriate field which goes after the "Path to the file" label. See figure "Path to the report".

Figure "Path to the report"

  "General" tab

"General" tab contains basic options of the report. The user can also define the contents of the report by using the "contents area" (under development). See figure "General tab".
Options:

  • Generate table of contents - if user selects this option, table of contents will be included in the report (by default it is selected).
  • Generate diagrams - if user selects this option, each diagram description will contain the picture of this diagram (by default it is selected).

Figure "General tab"


  "Title page" tab

"Title page" tab contains options related to the title page of the report. See figure "Title page tab". If the user wants to include title page to the report, "Generate title page" option should be selected (by default it is selected).
Additional options which can be defined:

  • Logo image path - by defining the path to the image file (*.jpg, *.jpeg, *.gif, *.png, *.tif or *.tiff file), the user includes this image file to the title page of the report. The path can be entered manually or by using ". . ." (browse) button.
  • Title - this option defines the title of the report.
  • Author name - this option defines the name of the author or organisation which will be displayed on the title page.

Figure "Title page tab"


  Generate report

After defining all report options, the user can generate report by pressing the "Generate" button.
If the user doesn't want to generate report, "Cancel" button should be pressed, but all report options, which was defined by the user will be saved.
