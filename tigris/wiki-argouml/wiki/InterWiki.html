Wiki: InterWiki

Edit this page | Links to this page | Page information | Attachments | Refresh page

 

InterWiki links are a quick way to refer to information in other wikis, without having to remember full URLs. To create an InterWiki link, just use the abbreviated name of the remote wiki, followed by the page name to link to. An example:

MoinMoin:WhyWikiWorks links to WhyWikiWorks.

InterWiki links are not rendered differently when the remote page does not exist, and no checking is done to ensure that you followed the rules for WikiNames of the remote wiki. See the InterWiki page on MeatballWiki for further details.
