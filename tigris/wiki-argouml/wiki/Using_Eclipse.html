Wiki: Using Eclipse

Eclipse is the IDE used by ArgoUML developers to do the Java (and XML) development. It has a built-in connection to the Version Control System and there are files prepared in to Version Control System to set up the projects and dependencies automatically. There are some things you need in order to work. This is described in Setting up Eclipse 3.

You need to have the core project downloaded and installed in your Eclipse.
