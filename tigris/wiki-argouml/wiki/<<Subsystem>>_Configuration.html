Purpose
Store and retrieve all configurable aspects of the ArgoUML application
Located

org.argouml.blablabla

Layer

<<Layer>> Low-level subsystems

It handles all configuration that is stored in the ".argouml" directory under the user's home directory in the file named "argo.user.properties".

Not inside the scope of this package are all the configuration settings that are stored in the project, i.e. in the .zargo file.
