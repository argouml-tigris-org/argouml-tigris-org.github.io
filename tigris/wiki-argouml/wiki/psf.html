Team Project Set Files for ArgoUML

These files are used to automatically set up Eclipse for ArgoUML development. For details on how to use these files, see Setting up Eclipse 3.

argouml-core-projectset.psf

Main file for developing ArgoUML.

argouml-command-line-build-projectset.psf

Command-line-build support for Eclipse working copies. Requires argouml-core-projectset.psf.

argouml-modules-projectset.psf

File with all modules. Requires argouml-core-projectset.psf.

argoeclipse_projectset.psf

For developing ArgoEclipse. Requires argouml-core-projectset.psf.

argoprint_projectset.psf

For developing ArgoPrint. Requires argouml-core-projectset.psf.

gef-swidgets-toolbar-projectSet.psf

For downloading the source of some of the libraries used by ArgoUML.

argouml-doc-projectset.psf

For working with the documentation & website.
