Wiki: Docbook editing For Eclipse Users

If you use Eclipse, then the XML editor that comes with the Eclipse Tools Platform (WTP) project allows comfortable editing.

Regretfully, the docbook-validation of this editor (as any other suitable editor that has been evaluated) currently does not support the way the quickguide is divided into seperate XML files. The manual consists of one single XML file, so it does not have this problem.
