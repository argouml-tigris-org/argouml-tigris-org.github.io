Wiki: Working with the Wiki

A Wiki is a collaborative site. Anyone can contribute and share.

If you are a member of the ArgoUML project you can:

  • Edit any page by pressing Edit this page at the top of the page

  • Create a link to another page with joined capitalized words (like WikiSandBox) or with ["quoted words in brackets"]

  • Search for page titles or text within pages using the Find page

  • See HelpForBeginners to get you going, HelpContents for all help pages

  • SyntaxReference: quick access to wiki syntax

In the project we use the following:

  • WikiConventions: describes how to write in the ArgoUML wiki and how we use the wiki in the ArgoUML project

  • Find tasks within the wiki on the ToDo page.
