Mission

Main goal of project is to provide full roundtrip engineering of Ruby code. Currently I have implemented simple Ruby code generation. In future I would like to to improve generator and implement reverse engeering module for Ruby. Project aims to provide environment for fast prototyping using Ruby language and perhaps Ruby on Rails framework

Main goals

  • Generate Ruby code from a UML model
  • Import Ruby source and build a UML model based on the source
